# magic-wood

Interactive climbing route map for Magic Wood boulder area. Features an SVG-based map with pan/zoom capabilities, difficulty filtering, and offline support.

## Project Setup

```sh
npm install
```

### Development

```sh
npm run dev
```

### Build

```sh
npm run build
```

### Lint

```sh
npm run lint
```

### Test

```sh
# Run tests in watch mode
npm run test

# Run tests once
npm run test:run

# Run tests with UI
npm run test:ui
```
